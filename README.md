# project3.github.io

Description:
A GUI-based Tic Tac Toe game for two players. It features an interactive board, winner detection with highlighting, and a reset option to play again.

Requirements:
Python 3.x
Libraries:
tkinter (built-in)

Instructions:
Setup:

Save the provided code as tic_tac_toe.py.
Run the script using python tic_tac_toe.py.
Gameplay:

Player X and Player O take turns by clicking the grid buttons.
The game checks for a winner or draw after each move.
Winning rows, columns, or diagonals are highlighted.
Click "Reset Game" to start a new round.
Enhancements:

Add a single-player mode with an AI opponent.
Allow players to customize symbols (e.g., emojis).
